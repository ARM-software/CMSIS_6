Description

The ISR shows whether an IRQ, FIQ, or external abort is pending.

Bits Name Function
[31:9] - Reserved.
[8] A External abort pending bit.
[7] I IRQ pending bit.
[6] F FIQ pending bit.
[5:0] - Reserved.

Consider __get_IFSR to access this register.
