Description

Codes to configure the CAN driver.

The various CAN control codes define:

  • CAN Identifier specify CAN identifier. Refer to ARM_CAN_ObjectConfigure.
  • CAN Operation Codes control CAN interface operation. Refer to ARM_CAN_Control.
  • CAN Bit Timing Codes specify CAN bit rate and timing. Refer to ARM_CAN_SetBitrate.
  • CAN Bus Communication Mode specify CAN bus operating mode. Refer to ARM_CAN_SetMode.
  • CAN Filter Operation Codes specify CAN filter operations. Refer to ARM_CAN_ObjectSetFilter.
  • CAN Object Configuration Codes specify CAN object configuration modes. Refer to ARM_CAN_ObjectConfigure.
